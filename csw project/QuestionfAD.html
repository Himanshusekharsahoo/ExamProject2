Exam Form For ALGORITHMS DESIGN 1

1. Consider an array A containing N integers. What is the time complexity of finding the smallest element in A using a linear search?


2. You are given a binary search tree (BST) containing N nodes. How many comparisons are needed in the worst case to search for a specific element in the BST?


3. What is the most efficient sorting algorithm for an array of N distinct integers if the order of the elements does not matter (only unique elements are important)?


4. Given a directed acyclic graph (DAG) with N nodes and M edges, what is the time complexity of finding the topological order of the nodes using Kahn's algorithm?


5. What is the space complexity of the recursive implementation of the Fibonacci sequence function?
